# PDF Chatbot

A Streamlit-based PDF Chatbot that allows users to upload PDF documents and ask questions about their content.

## Features
- Upload one or more PDF files
- Extract text from PDFs
- Ask questions about PDF content
- AI-powered responses using LangChain and LLMs
- Simple Streamlit user interface

## Create Virtual Environment:

```bash
python -m venv myenv
myenv/scripts/activate
```
## Run the Application

```bash
streamlit run app.py
```
